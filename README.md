# Lycium - Trình tạo thẻ Tam Quốc Sát trực tuyến

Vui lòng truy cập [Lycium Trình tạo thẻ Tam Quốc Sát](https://catscarf.github.io/) để sử dụng ứng dụng này.

Chỉ dành cho mục đích học tập và trao đổi. Hãy tặng một ngôi sao (Star) để ủng hộ dự án này nhé~

## Cách cài đặt

```bash
git clone https://github.com/CatScarf/Lyciumaker.git
cd Lyciumaker
npm install
npm run dev
```

## Nhật ký cập nhật

### Nhật ký cập nhật 2.5

Nâng cao độ chính xác khi vẽ trên màn hình độ phân giải thấp, tránh chữ bị mờ.

Tối ưu hóa trải nghiệm kéo thả trong tính năng ghép chữ, hiện đã có thể kéo thả bình thường trên điện thoại.

Tăng khoảng cách giữa các kỹ năng.

### Nhật ký cập nhật 2.4

Tối ưu hóa mạnh mẽ hiệu suất vẽ, giải quyết vấn đề quá tải trên thiết bị di động.

Điều chỉnh cỡ chữ mặc định, hiện tại tên võ tướng có 3 chữ sẽ không bị quá lớn.

### Nhật ký cập nhật 2.3

Ảnh minh họa có thể kéo thả.

Loại bỏ chữ "Giới" (界) hiển thị không chính xác.

Tối ưu hóa một số mã nguồn.

### Nhật ký cập nhật 2.2

Tách các tham số vẽ thành tệp cấu hình độc lập, chuẩn bị cho việc thêm giao diện UI mới sau này.

Tối ưu hóa một số chi tiết nhỏ.

### Nhật ký cập nhật 2.1

Sửa lỗi lệch tên kỹ năng trên trình duyệt Safari.

Sửa lỗi khung nền kỹ năng quá hẹp.

### Nhật ký cập nhật 2.0

Sử dụng Vue để tái cấu trúc toàn bộ dự án và thiết kế lại giao diện UI.

Thêm tính năng ghép chữ, tên võ tướng không còn lo bị thiếu chữ.

Tăng đáng kể tốc độ tải, giảm tải truyền tải mạng.
